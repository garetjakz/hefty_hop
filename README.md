# Hefty Hop — 80s-style infinite platformer starring the Hefty Hippos

Play: open `index.html`, or deploy this folder to any static host.

- Procedurally generated levels, one-hit arcade rules, high-score table
- Sprites & backgrounds derived from the Hefty Hippos NFT collection (8,888 hippos)
- Keyboard: arrows/WASD move, Space jump (hold = higher), Down+Jump = Belly Bump, X/K fire, R restart
- Touch controls on mobile

Built with plain HTML5 canvas — no build step, no dependencies.
